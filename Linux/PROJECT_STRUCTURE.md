## Data Management

### Offsets Folder:
All offset scan results and SDK are organized in `offsets/`:
- `phantom_offsets.json` - Current active offsets
- `phantom_offsets_backup_*.json` - Timestamped backups (every 5 seconds)
- `sdk/` - Game SDK & engine data (moved from root)

### Documentation & Research:
All documentation and research in `docs/`:
- `research/` - Research compilation & patterns (moved from root)
- `BLOODSTRIKE CHEAT - DOCUMENTATION/` - Full documentation

### Backups Folder:
- Old configuration files
- Deprecated overlay versions
- Inventory backups

### **Logs Folder:**
- Application logs
- Debug outputs
- Performance metrics

## 🚀 **Quick Start**

```bash
# Launch professional suite
./start.sh

# Or launch components individually
python phantom_strike_pro.py          # All components
python phantom_overlay_pro.py        # Overlay only
python debug_components.py           # Test components
```

## 🎮 **In-Game Menu**

Press **INSERT** to open the professional menu with 5 categories:

1. **ESP** - Box ESP, health bars, armor, distance, weapons, names
2. **Aimbot** - Real/AI aimbot, FOV, smoothness, prediction
3. **Memory** - Auto scan, continuous scanning, ML prediction
4. **Skin Changer** - Enable skins, individual weapons, StatTrak™
5. **Options** - Anti-cheat, stealth mode, FPS limit, stats

- `backups/*.json` - Configuration backups
